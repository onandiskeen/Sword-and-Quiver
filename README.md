# Sword and Quiver

## Introduction

Sword and Quiver is a 2D platformer built entirely in Java.
Switch instantly between a sword fighter for close combat and an archer for ranged attacks. Defeat enemies, adapt your playstyle, and fight your way through each level to progress.
